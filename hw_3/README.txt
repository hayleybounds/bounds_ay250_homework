Variables needed for monty:
PATH_TO_API_KEY: the path to the json file containing the credentials for the google speech api.
	I followed the instructions here: https://github.com/GoogleCloudPlatform/python-docs-samples/blob/master/speech/cloud-client/README.rst
They tell you how to download the necessary things and direct you to where to get the credentials.

GMAIL_USER: username that will send the email
GMAIL_PW = password for that email
GMAIL_RECIPIENT = who to send the email to
SMTP_SERVER = what server to use, I used "smtp.gmail.com"
SMTP_NUMBER = the port for that server, I used 587
